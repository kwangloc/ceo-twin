# Project Status Skill

Use this skill to assess project health and diagnose delays or blockers.

## Steps

1. **Identify the objective** — What is the project trying to deliver and by when?
2. **Find blockers** — What dependencies are missing, late, or at risk?
3. **Identify owners** — Who is responsible for each blocker or decision?
4. **Assess impact** — How does this affect the timeline, budget, or stakeholders?
5. **Summarize clearly** — Produce a director-level summary: status, risks, recommended actions.

## Output Format

- **Status**: On track / At risk / Delayed / Blocked
- **Key blockers**: (list)
- **Owners**: (name → responsibility)
- **Recommended actions**: (list with urgency)
